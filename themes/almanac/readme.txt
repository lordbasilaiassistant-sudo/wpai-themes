== Description ==

Almanac is a WordPress theme for people who tend their writing like a living
thing — a personal wiki, a knowledge base, a slowly-growing collection of
interlinked notes. It lays a warm, paper-like canvas over a faint dot grid (the
feel of a dot-grid notebook), surfaces tags on every card, and stamps each note
with when it was Planted and when it was Last tended — so a reader can tell at a
glance which ideas are evergreen and freshly watered.

The whole theme leans into cross-linking. Internal links, the nav, entry
titles, and in-article links all share one connective gesture: a dotted
verdigris thread that grows in (and goes solid) on hover and focus, so following
a link always feels like reaching for a thread between ideas.

It ships with real CSS — custom properties, a modular type scale, a responsive
two-column reading layout, and accessible focus states — and uses only
system/web-safe fonts, so it loads instantly with no external requests.

Features:

* Garden home: a featured "seedling" (your freshest note) over an elegant bed
  of note cards, each with its cover, category patch, tags, and tended stamps
* Evergreen metadata: "Planted" (publish) and "Last tended" (modified) stamps,
  shown only when a note has actually been revised
* Visible tags everywhere — compact #hash tag rows on cards, a "Threads" footer
  on single notes, and a tag-cloud widget styled as a literal tag patch
* Connective cross-link hover: a dotted verdigris thread that grows in on
  hover/focus across the nav, titles, body links, comments, and widgets
* Featured images on the index, single, and pages — with a graceful verdigris
  "garden paper" placeholder when a note has none, so the rhythm never breaks
* Comfortable reading measure, serif body type, balanced headings, and a small
  leaf-tick before article H2s
* Two-column layout with a sticky, widgetised sidebar ("the stacks")
* Threaded comments ("marginalia") and a polished comment form
* Accessible, responsive mobile navigation with a no-JS fallback
* Skip link, visible focus-visible states, semantic landmarks, AA contrast,
  and full prefers-reduced-motion support
* Signature motion: a gentle "growing" reveal (blocks rise and grow up from the
  baseline) plus a connective "vine sprout" that draws itself in beside titles
* Slim, vine-coloured reading-progress bar on single notes
* Live Customizer color controls (Accent, Background, Surface) with instant
  preview; accent shades derive via color-mix()
* Native WPAI companion-plugin integration via wpai_entry_top / wpai_entry_bottom
* Block editor color palette and font sizes via theme.json
* Custom logo, primary + footer menus, and a styled 404
* Translation-ready (text domain: almanac). No external fonts, no tracking.

== Installation ==

1. In wp-admin go to Appearance > Themes > Add New > Upload Theme.
2. Choose almanac.zip and click Install Now.
3. Click Activate.
4. Set a Primary menu under Appearance > Menus, add widgets under
   Appearance > Widgets (a Tag Cloud looks great in the stacks), and give your
   notes featured images and tags for the best look.
