== Description ==

Emporium is a premium storefront theme for modern online shops, given away free.
It is built to feel like a paid Shopify build — a confident editorial hero, a
featured-product grid, shoppable category tiles, social proof, a promo band, and a
journal — in a warm boutique palette of charcoal ink on porcelain with a quiet
evergreen accent.

It is designed to pair with two free companion plugins, **Till — Commerce** (the shop
engine: products, a slide-in cart, checkout) and **Keepsake** (a wishlist). Together
they give you a real store with a sticky header cart count, an add-to-cart drawer, and
a saved-items page out of the box — no page builder, no monthly fee, no external fonts.

* **A storefront homepage.** A hero with a clear call to action and a floating
  best-seller tag, a trust-signal strip, a "new arrivals" product grid, shop-by-category
  tiles built from your real products, a promo band, a from-the-journal teaser, and a
  newsletter strip.
* **A sticky store header.** Centered navigation, a slide-down search, a wishlist link
  with a live saved-count, and a cart button whose count updates instantly as shoppers
  add items. It condenses with a subtle shadow as the page scrolls.
* **Beautiful product pages.** Till injects a clean two-column product layout (gallery,
  price, rating, quantity, add-to-cart, related products); Emporium frames it with a
  breadcrumb and full-width layout.
* **Shop & category pages.** A filterable, sortable, paginated shop grid and elegant
  product-category archives, all styled to match.
* **A journal.** A tasteful blog for stories, lookbooks, and guides, with a two-column
  list and an optional sidebar.
* **Recolour everything from the Customizer.** Three controls — Accent, Background, and
  Ink — drive the whole theme *and* the storefront's buttons and cart, live.
* **Graceful without the plugins.** Activate Emporium on its own and it still reads as a
  polished landing-page-and-blog theme; add Till and the shop sections light up.
* **Fast, accessible, self-contained.** System fonts only, responsive from 320px up,
  keyboard-friendly, reduced-motion aware, and no external requests or trackers.

== Installation ==

1. In wp-admin go to Appearance > Themes > Add New > Upload Theme.
2. Choose emporium.zip, Install Now, then Activate.
3. Accept the admin prompt to install the free companion plugins (Till — Commerce and
   Keepsake), or install them from the WPAI Themes gallery. Till creates your Shop,
   Cart, and Checkout pages and stocks a demo catalog automatically.
4. Set a static front page (Settings > Reading) to use the storefront homepage, add
   "Shop" to your menu (Appearance > Menus), and you're live.

== Frequently Asked Questions ==

= Do I need a commerce plugin? =

For a working shop, yes — install the free **Till — Commerce** plugin (Emporium prompts
you to). On its own, Emporium is still a complete, polished landing-page-and-journal
theme; the store sections simply appear once Till is active.

= Is it really comparable to a paid Shopify theme? =

That's the goal: a sticky cart header, an add-to-cart drawer, a featured grid, category
tiles, a wishlist, and a checkout — the storefront essentials — built to a premium bar
and given away under the GPL.

= How do I get the storefront homepage? =

Set a static front page under Settings > Reading. Emporium's front-page template renders
the full storefront; your "Posts page" becomes the Journal.

= Can I change the colors? =

Yes — Appearance > Customize > Colors & Style. The Ink control also recolours the
store's buttons and cart so everything stays in sync.

= Does it work in WordPress Playground? =

Yes — the live preview on the WPAI Themes gallery runs Emporium with Till and Keepsake
in Playground, fully self-contained.
